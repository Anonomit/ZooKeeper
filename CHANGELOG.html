2.2.0

New Features

  • Shaman's Ghost Wolf is now used. It should only be used while in combat, or when no mounts are available.
  • Goblin Rocket Pack is used (when available) in Icecrown Citadel.

Fixes

  • Fixed mounting for some classes with multiple hotbars, such as priests, warriors, and rogues.
  • Fixed an issue with higher level druids that could cause the mounting script to misbehave, usually by not exiting shapeshift forms.
