3.0.0

New Features

  • Classic Era compatibility.
  • Favorites can now be managed in the options menu (/zk).
  • Added many new zone-specific options. Configure usage in options menu. Current list is below.
    • Molten Core: Eternal Quintessence and Aqual Quintessence.
    • Blackwing Lair: Unequip/equip ranged weapon for Nefarion (hunters only).
    • Karazhan: Blackened Urn for Nightbane (only during TBC).
    • Serpentshrine Cavern: Tainted Core for Lady Vashj.
    • The Eye: All legendary weapons for Kael'thas (equip and use).
    • Battle for Mount Hyjal: Tears of the Goddess for Archimonde.
    • Black Temple: Naj'entus Spine for High Warlord Naj'entus.
    • Oculus: All dragon mounts.
    • Ulduar: Magnetic Core for Mimiron.
    • Icecrown Citadel: Goblin Rocket Pack for Icecrown Gunship Battle (equip and use).

Fixes

  • Added missing libraries.
  • Reverted pet summoning change, since some pets can't be summoned by name. Pets are no longer be summonable during combat in Wrath.

  • Fixed an issue with higher level druids that could cause the mounting script to misbehave, usually by not exiting shapeshift forms.

2.0.0

New Features

  • Mounting as a druid now uses an appropriate shadpeshift form if it makes more sense (combat, swimming, moving, etc.). This is currently not configurable.

Changes

  • Old slash commands no longer work, due to the Collection Log.
    • Mounting is now done with /click ZKM (ZooKeeper Mount).
    • Summoning a pet is now done with /click ZKP (ZooKeeper Pet).
    • It is no longer possible to specify a certain type (ground/fly/swim) or speed of mount. You may still set favorite mounts and the addon will use them unless a faster one is found.
  • Favorite pets have been reset due to Blizzard moving to a different way of storing pets. Favorite mounts have been preserved.

Fixes

  • Updated to work with the new Collection Log.
